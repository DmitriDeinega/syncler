Item 2 — FastAPI broker: RED

- Missing test: valid-shape envelope with a 32-byte invalid/low-order `ephemeral_pubkey` must return `401`, not `500`.
- Packaging footgun: new broker tests import FastAPI, but `sdk-python[dev]` does not include the `broker` extra. A clean dev install may not be able to run the full SDK test suite.
